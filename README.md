# Chad Chat

Um chat desenvolvido em React utilizando o Firebase para comunicação assíncrona e em tempo real com o envio e recebimentos de mensagens, com uma tematica CHAD

## 🚀 Começando

Entrar no link para visualização do conteúdo ---> [LINK](https://bruno-gdos.github.io/Chad-Chat/)

Logo em seguida irá ser necessária a criação de uma conta, utilizando qualquer email, login e a necessidade de uma imagem aleatória, em seguida procurar na barra de pesquisa pelo nome de algum usuário em nosso banco de dados para iniciar a busca.

### 📋 Pré-requisitos

Um nevegador web e uma conexão com a internet
